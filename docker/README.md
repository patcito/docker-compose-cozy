Documentation
=============

Here is a few hints about Docker.

List the containers
-------------------

    $ docker ps

You may want to add the `-a` argument to list all of them.

    $ docker ps -a

Remove a container
------------------

    $ docker rm <container_id>

List the images
---------------

    $ docker images -a

You may want to add the `-a` argument to list all of them.

Remove an image
---------------

    $ docker rmi <image_id>
